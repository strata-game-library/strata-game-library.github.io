# Strata Documentation

This is the official documentation site for Strata, a procedural 3D graphics library for React Three Fiber.

## Getting Started

To get started with Strata, check out the [Getting Started guide](https://strata.game/getting-started).

## Contributing

This site is built with [Astro](https://astro.build/) and the [Starlight](https://starlight.astro.build/) theme. To contribute, clone the repository and run `npm install` to install the dependencies. Then, run `npm run dev` to start the development server.
